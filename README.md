# 7月投顾与T0奖励惩罚核算台账

这是上海水清南路营业部 2026 年 7 月投顾与 T0 奖励辅助核算网页。

## 使用方式

1. 打开网页后选择员工，录入投顾客户、T0客户和必要的订单/创收数据。
2. 系统会自动刷新右侧核算结果和计算过程。
3. 确认无误后点击“写入台账”，该员工明细才会进入员工记录和 Excel。
4. 顶部“以 Excel 形式导出台账”会导出员工汇总和每位员工的核算工作表。

## 数据保存

- 数据保存在当前浏览器本地，不上传服务器。
- 更换浏览器或清除浏览器数据后，本地台账会消失。
- 手工更正先暂存，随“写入台账”一起进入员工记录和 Excel。

## 7月规则摘要

- 营业部投顾：新增投顾客户 50 元/户；新增客户创收按 50/100/150 元最高档奖励；创收目标 4000 元，创收不超过 10000 元时得分最高 8 分，超过 10000 元不封顶。
- 营业部T0：新增T0有效户 200 元/户。
- 营业部剥离：投顾阶段总分不足 10 分时，按总分整数档与 10 分比较，每少 1 档剥离 2 户。
- 上海分公司投顾：按 7 月 PDF 的订单净增、新增客户数、创收分层、订单破冰和服务满月复核规则核算。
- 上海分公司T0：沿用 6 月口径，新增T0客户交易基础奖、交易量补充和有效户阶梯奖励。

## 部署到 GitHub Pages

将生成包中的 `index.html` 放到仓库根目录，在 GitHub 仓库 Settings -> Pages 中选择部署分支即可。
